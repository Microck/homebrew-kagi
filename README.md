# homebrew-kagi

Homebrew tap for the [Kagi](https://kagi.com) CLI - a powerful AI-assisted search engine.

## Install

```bash
# Add the tap and install kagi
brew tap Microck/kagi
brew install kagi
```

## Upgrade

```bash
# Update Homebrew and upgrade kagi
brew update
brew upgrade kagi
```

## Packages

| Package | Description                    |
|---------|--------------------------------|
| kagi    | Kagi Search CLI - AI search    |

## Requirements

- [Homebrew](https://brew.sh) - macOS/Linux package manager
- macOS or Linux operating system
