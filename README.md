# homebrew-kagi

Homebrew tap for the [`kagi`](https://github.com/Microck/kagi-cli) CLI — an agent-native Rust CLI for Kagi subscribers with JSON-first output.

## Install

```bash
brew tap Microck/kagi
brew install kagi
```

## Upgrade

```bash
brew update
brew upgrade kagi
```

## Uninstall

```bash
brew uninstall kagi
brew untap Microck/kagi
```
